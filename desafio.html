Criar Uma Página Web Utilizando as Tags Aprendidas

O que é TAG?

Tag é uma marcação que é utilizada para definir o início e o fim de um elemento em uma página web.

Exemplos de Tags

Tag <h1> até <h6>

São usadas para definir cabeçalhos de diferentes níveis de importância, sendo <h1> o mais importante e <h6> o menos. São úteis para estruturar o conteúdo da página e melhorar a acessibilidade.

Tag <p>

Usada para definir parágrafos de texto. É uma das tags mais básicas e comuns para organizar o conteúdo de texto em uma página HTML.

Tag <mark>

Usada para destacar parte do texto, geralmente com um fundo amarelo. É útil para chamar a atenção do leitor para informações importantes.

Tag mark: Texto em destaque dentro de um parágrafo.

Tag <small>

Define um texto como menor e com uma fonte mais discreta. É útil para notas de rodapé ou informações menos relevantes.

Tag small: Texto pequeno dentro de um parágrafo.

Tag <i>

Utilizada para colocar o texto em itálico. É mais usada para destacar palavras ou frases em um contexto semântico.

Tag Itálico: Texto em itálico dentro de um parágrafo.

Tag <u>

Usada para sublinhar o texto. No entanto, em termos de estilo, é geralmente desencorajada em favor de estilos CSS para evitar confusão com links.

Tag underline: Texto sublinhado dentro de um parágrafo.

Tag <strong>

Define o texto como forte ou importante, geralmente exibido em negrito. É usado para enfatizar a importância de certas palavras ou frases.

Tag negrito: Texto em negrito dentro de um parágrafo.

Tag <ol>

Cria uma lista ordenada, onde cada item é numerado. Exemplo abaixo:

Lista de Tarefas

  1. Estudar HTML
  2. Fazer exercícios de CSS
  3. Praticar JavaScript

Tag <ul>

Cria uma lista não ordenada, onde cada item é marcado com um bullet. Exemplo abaixo:

Lista de Compras

  • Maçãs
  • Bananas
  • Laranjas
  • Uvas

Tag <li>

Usada dentro de <ol> ou <ul> para definir cada item da lista.

Tag <a>

Cria um hyperlink, permitindo que os usuários cliquem em um texto ou imagem para navegar para outra página ou recurso na web.

Utilizar novas tags sugeridas:

Tag <Sub>

Texto subscrito: Esta tag é usada para definir texto como subscrito, exibindo-o ligeiramente abaixo da linha de base normal.

Tag <Sup>

Texto sobrescrito: Esta tag é usada para definir texto como sobrescrito, exibindo-o ligeiramente acima da linha de base normal.

Tag <Blockquote>

Citação em bloco: Esta tag é usada para criar uma citação em bloco, geralmente com um recuo em relação ao restante do texto.

Tag <Font>

Texto com cor vermelha: Esta tag (obsoleta em HTML5) é usada para definir a cor do texto. No entanto, é mais recomendado usar CSS para estilizar o texto.

Tag <Del>

Texto tachado: Esta tag é usada para definir o texto como tachado, indicando que ele foi removido ou deletado.

Tag <Abbr>

WWW: Esta tag é usada para definir uma abreviação ou acrônimo, fornecendo uma descrição completa quando o usuário passa o mouse sobre a abreviação.
